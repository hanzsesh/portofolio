Apa Itu CSS?

CSS (Cascading Style Sheets) adalah bahasa yang digunakan untuk mendesain dan mengatur tata letak halaman web. Tanpa CSS, halaman web kita akan tampak seperti sandwich tanpa isi: tidak menarik!

Struktur Dasar CSS

Sebelum kita mulai, yuk lihat struktur dasar CSS. Ingat, ini seperti resep masakan—setiap elemen harus tepat:

selector {
  property: value;
}

Pemilihan Elemen

Dengan CSS, kita dapat memilih elemen HTML berdasarkan tag, kelas, atau ID. Bayangkan kamu adalah penata rias yang memilih produk terbaik untuk wajahmu:

p {
  color: blue;
}
.classname {
  font-size: 20px;
}
#idname {
  background-color: red;
}

Properti CSS Umum

Beberapa properti CSS yang sering digunakan adalah:

color: Mengatur warna teks. (Supaya tidak monoton!)
background-color: Mengatur warna latar belakang. (Agar tidak membosankan!)
font-size: Mengatur ukuran font. (Supaya tidak bikin mata pegel!)
margin: Mengatur jarak luar elemen. (Agar tidak berdesakan!)
padding: Mengatur jarak dalam elemen. (Supaya nyaman dilihat!)

Box Model

Setiap elemen dalam HTML memiliki box model yang terdiri dari margin, border, padding, dan konten. Ini penting untuk mengatur tata letak elemen—kayak memilih baju yang pas:

box-sizing: border-box;
margin: 10px;
border: 1px solid black;
padding: 5px;
