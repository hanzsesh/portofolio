Apa Itu Python?

Python itu kayak sahabat setia, selalu ada saat kamu butuh! Dia juga nggak neko-neko, karena sintaksisnya yang mudah dimengerti. Python itu bahasa pemrograman yang banyak dipakai di dunia. Tanpa Python, banyak orang bakal bingung, seperti saat kamu lupa password Wi-Fi!

Struktur Dasar Python

Yuk, lihat struktur dasar dari program Python:

print("Halo, Dunia!") # Menampilkan teks ke layar

Variabel dan Tipe Data

Variabel itu kayak kotak penyimpanan. Kamu bisa simpan berbagai jenis data di dalamnya. Misalnya:

angka = 10
nama = "Hanz"
print(nama, "memiliki angka:", angka)

Pengkondisian

Python bisa bikin keputusan. Misalnya, kalau kamu lapar, makanlah! Contohnya:

if angka > 5:
   print("Angka lebih besar dari 5!")
else:
   print("Angka 5 atau lebih kecil!")

Looping

Looping itu seperti ketika kamu berulang kali menjawab "sudah makan?" saat lebaran. Yuk, kita lihat contohnya:

for i in range(5):
   print("Ini pengulangan ke-", i)

Fungsi

Fungsi adalah kumpulan kode yang bisa dipanggil berkali-kali. Seperti kamu yang terus-menerus bercerita tentang makanan favorit:

def cerita_makanan():
   print("Saya suka pizza!")

cerita_makanan() # Memanggil fungsi
