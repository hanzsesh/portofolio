Apa Itu JavaScript?

JavaScript adalah bahasa pemrograman yang memberi kehidupan pada website. Tanpa dia, website itu kayak sayur tanpa garam. Sehat, tapi kurang nendang!

Struktur Dasar JavaScript

JavaScript biasanya disisipkan dalam tag <script> di dalam HTML. Mari kita lihat contohnya:

<script>
  console.log('Halo, Dunia!');
</script>

Variabel

Variabel itu seperti tasmuâ€”tempat menyimpan barang (data) penting. Kita bisa mendeklarasikan variabel dengan kata kunci let, const, atau var. Ayo, simpan data dengan bijak!

let nama = 'Hanz';
const umur = 25;
var kota = 'Jakarta';

Fungsi

Fungsi adalah sekumpulan kode yang bisa dipanggil berulang kali. Bayangkan kalau kamu bisa memanggil makanan favoritmu terus-menerus!

function sapa() {
  console.log('Selamat datang!');
}

Event Handling

JavaScript juga memungkinkan kita untuk menangani event, seperti klik tombol. Seperti saat kamu klik "next episode" di drama favorit!

<button onclick="sapa();">Klik Saya</button>
